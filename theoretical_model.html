Theoretical model

This theoretical model represents entities and relationships at a first level of abstraction with respect to the E / R model. This means that each instance has been replaced with its own class, but always using natural language and not controlled vocabularies. In particular, the properties related to production, ie the place, the time and the creator, have been summarized in a single "Production" entity.
